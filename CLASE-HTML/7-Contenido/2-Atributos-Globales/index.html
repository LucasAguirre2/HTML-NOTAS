Atributos globales:

Atributo Id

Es una identificacion unica, se utiliza mucho para identificar una etiqueta.

Ejemplo:

No hay nadie como yo

Atributo Class

Es parecido a Id con la diferencia que puedes repetirlo en muchas etiquetas, es mas como una identificacion para un grupo

Ejemplo:

Unidos somos mas

Atributo contenteditable

Se usa para editar el contenido

Ejemplo:

Puedes editar este contenido

Atributo lang

Indica el idioma del contenido

Ejemplo:

Esto esta escrito en español

Atributo dir

dir: significa direccion. dir:="rtl" de derecha a izquierda
dir:="ltr" que es de izquierda a derecha
