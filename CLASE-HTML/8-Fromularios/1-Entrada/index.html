Etiqueta input

La etiqueta input en HTML se utiliza para crear elementos interactivos en un formulario web. Se utiliza para recopilar información del usuario, como texto, números, fechas, opciones de selección y más.

Atributos Type="Password"

se utiliza para ingresar contraseña

Atributo type="search"

se utiliza como un buscador

Atributo type="tel"

se utiliza para numero de telefono

Atributo type="date"

se utiliza para fechas

Atributo type="color"

Sirve para colores

Atributo input="file"

Sirve para colocar archivos

Atributo input="checkbox"

agrega un checkbox
